Responsive Reading Area

This test page is built for mobile. It keeps the page layout stable while allowing you to adjust only the text size and line height inside this content block. Scroll with one finger as usual. Use a two-finger pinch in this area to make the text smaller or larger.

Small Text Sample

This smaller paragraph helps verify that relative sizing still looks clean when the base content font changes. Line height is recalculated together with the font size, so text should remain readable and not look cramped.

Normal Text Sample

The main reading size is controlled by a single CSS variable. Everything inside this article uses relative units, so headings, chips, and body text all scale in proportion. That makes the content reflow naturally according to the mobile screen width.

Large Text Sample

This larger paragraph is useful for checking readability, wrapping, and vertical rhythm. The goal is to adjust typography only, not browser zoom. That means the viewport stays stable while the content becomes easier or harder to read based on your gesture.

Long Paragraph Test

Mobile reading experiences often fail when text size changes without matching line height. In this demo, line height is linked to the active font size so that each paragraph keeps comfortable spacing. The page also updates itself on orientation change and window resize, which helps keep the content ordered on different screen sizes.
