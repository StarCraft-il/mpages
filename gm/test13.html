Implementation Notes

This version avoids extra APIs that are unnecessary in WebView and uses direct touch event tracking with requestAnimationFrame. It also prevents browser zoom by using a locked viewport and intercepting two-finger movement.

If you host this inside a native Android app, make sure the WebView itself is not configured to force zoom controls or automatic text zoom, otherwise the app-level settings may override the page behavior.
